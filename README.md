# Crossing Over into God's Promised Land

A narrative Bible study site on crossing over from Deuteronomy to the fall of Jericho.

Built as a single scrollable page.

## Structure

- `index.html` — main page
- `style.css` — styling (dark navy + gold, Cormorant Garamond + Inter)
- `images/` — AI-generated illustrations

## View locally

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
